# v1.1.0 — 插件、同步与性能飞跃

发布日期：2026-06-01

> Muse v1.1.0 是一次重要的大版本更新，带来**插件化扩展系统**、**云端同步预览**和**全面性能优化**三大核心升级，让 Muse 从一款优秀的编辑器进化为可扩展的写作平台。

---

## ✨ 新增功能

### 🔌 插件化扩展系统

- 支持加载第三方扩展插件，可自定义编辑器功能、工具栏按钮和导出格式
- 提供扩展 API 文档，开发者可快速编写并分享自己的插件
- 内置扩展市场入口，一键安装社区热门扩展

### ☁️ 云端同步预览

- 新增多设备工作区状态同步能力
- 编辑进度自动保存至云端，切换设备无缝衔接
- 支持离线编辑，联网后自动合并变更

### 🔍 增强型搜索体验

- 全文搜索支持正则表达式匹配
- 搜索结果高亮显示上下文行
- 快捷键 `Ctrl+Shift+F` 唤起搜索面板

### 📤 导出格式升级

- 新增导出为 Word (.docx) 格式
- PDF 导出支持自定义页眉页脚
- 导出时自动生成目录索引

---

## ⚡ 性能大幅优化

- **启动速度提升约 40%**，冷启动时间缩短至 1s 以内
- **大文件（10MB+ Markdown）编辑流畅度显著改善**
- **编辑器内存占用降低 30%**，长时间运行更稳定
- TipTap 编辑器实例复用策略优化，减少重复初始化开销
- 文件监听（chokidar）配置调优，降低 CPU 占用
- 图片拖放处理流程异步化，避免 UI 卡顿
- CSS 动画启用 GPU 加速（will-change / transform）

---

## 🐛 修复

- 修复 Windows 下偶尔出现的中文输入法兼容问题
- 修复侧边栏文件树在深色模式下部分图标颜色过浅的问题
- 修复 Mermaid 图表在某些分辨率下渲染溢出的边界情况
- 修复任务列表嵌套超过 3 层时的缩进异常
