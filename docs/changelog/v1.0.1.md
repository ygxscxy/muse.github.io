# v1.0.1

发布日期：2026-05-27

## 新增

- **侧边栏打开文件入口**
  - Sidebar 顶部工具栏新增"打开文件"图标（File），可直接唤起文件选择对话框并打开 `.md`/`.txt`/`.markdown` 文件。

- **数学公式渲染链路打通**
  - `InlineMath` / `BlockMath` 的 `renderHTML` 现在会保留 `data-expression` 属性，确保保存/重新打开后公式内容不丢失。
  - 数学组件类型改用 `ReactNodeViewProps`，消除与 `ReactNodeViewRenderer` 的类型不兼容报错。

- **打开文件时自动预处理数学公式**
  - 引入 `markdownWithMathToEditorHtml`，在加载本地文件内容前先将 `$...$` / `$$...$$` 语法转换为编辑器可识别的数学节点标记。

## 优化与修复

- **生产环境禁用开发者工具**
  - 打包后的生产环境自动禁用底层开发者工具。
  - "切换开发者工具"菜单项仅在开发环境显示。

- **清理无用代码**
  - 侧边栏中去掉了冗余的文件读取返回值，避免 lint 警告。
