# 快速开始

## 安装

从[下载页面](/download)获取适合你平台的安装包。

### Windows

下载 `.exe` 安装包，双击运行即可。安装完成后，可在开始菜单找到 Muse。

### macOS

下载 `.dmg` 文件，将 Muse 拖入 Applications 文件夹。首次打开可能需要在「系统设置 > 隐私与安全性」中允许运行。

### Linux

下载 `.AppImage` 或 `.deb` 包。

- **AppImage**：赋予执行权限后双击运行，无需安装
- **deb**：使用 `dpkg -i muse.deb` 安装

## 界面概览

启动 Muse 后，你会看到简洁的三栏布局：

- **左侧边栏**：文件树，显示当前工作区的文件夹和文件结构
- **中间编辑区**：WYSIWYG 编辑器，所见即所得
- **顶部工具栏**：常用格式化按钮和文档操作

## 打开文件夹

1. 启动 Muse
2. 点击欢迎页面的「打开文件夹」按钮
3. 选择一个包含 Markdown 文件的文件夹作为工作区

> 提示：Muse 以文件夹作为工作区，你可以随时切换不同的项目文件夹。

## 创建文件

在工作区中，右键点击文件树区域：

- 选择「新建文件」创建 Markdown 文档
- 选择「新建文件夹」创建目录
- 支持右键重命名、删除、移动文件

## 编辑文档

点击文件树中的文件即可开始编辑。Muse 提供完整的 WYSIWYG 编辑体验：

- 使用顶部工具栏格式化文本（加粗、斜体、标题、列表等）
- 使用[快捷键](/guide/shortcuts)快速操作，大幅提升效率
- 拖拽图片直接插入，自动保存为相对路径
- 使用 `/` 快捷命令快速插入代码块、表格等元素

## 主题切换

Muse 支持深色和浅色主题，点击顶部工具栏的主题按钮即可一键切换，满足不同光线环境下的使用需求。
