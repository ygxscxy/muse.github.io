# 介绍

Muse 是一款基于 Electron 构建的现代化 WYSIWYG Markdown 编辑器，它结合了所见即所得的编辑体验与 Markdown 的简洁优雅。无论是撰写技术文档、整理学习笔记还是编写博客文章，Muse 都能让你专注于内容本身。

## 核心理念

### 所见即所得

无需在编辑器和预览之间频繁切换，输入即呈现。Muse 基于 TipTap 编辑器引擎，提供流畅的富文本编辑体验，让排版变得直观而自然。

### 纯文件优先

所有文档以纯 Markdown 文件形式存储在本地，不会锁定你的数据。你可以随时用任何文本编辑器打开和编辑 Muse 创建的文件，数据完全由你掌控。

### 扩展而不臃肿

支持代码块语法高亮、LaTeX 数学公式、Mermaid 图表等扩展能力，满足技术写作、学术研究和项目文档的全部需求，同时保持界面的简洁克制。

## 适用场景

- **技术博客**：代码高亮、数学公式、Mermaid 流程图，技术写作者的得力助手
- **学习笔记**：层级标题、任务清单、表格整理，知识管理井井有条
- **项目文档**：纯 Markdown 存储，方便版本控制和团队协作
- **日常写作**：专注模式、快捷键操作，让写作回归纯粹
