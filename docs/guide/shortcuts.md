# 快捷键

> 提示：macOS 用户请将 `Ctrl` 替换为 `Cmd（⌘）`，`Alt` 替换为 `Option（⌥）`。

## 编辑快捷键

| 快捷键         | 功能   | macOS       |
| -------------- | ------ | ----------- |
| `Ctrl+B`       | 加粗   | `⌘+B`       |
| `Ctrl+I`       | 斜体   | `⌘+I`       |
| `Ctrl+U`       | 下划线 | `⌘+U`       |
| `Ctrl+Shift+S` | 删除线 | `⌘+Shift+S` |
| `Ctrl+H`       | 高亮   | `⌘+H`       |

## 标题快捷键

| 快捷键       | 功能     | macOS   |
| ------------ | -------- | ------- |
| `Ctrl+Alt+1` | 一级标题 | `⌘+⌥+1` |
| `Ctrl+Alt+2` | 二级标题 | `⌘+⌥+2` |
| `Ctrl+Alt+3` | 三级标题 | `⌘+⌥+3` |
| `Ctrl+Alt+4` | 四级标题 | `⌘+⌥+4` |
| `Ctrl+Alt+5` | 五级标题 | `⌘+⌥+5` |
| `Ctrl+Alt+6` | 六级标题 | `⌘+⌥+6` |

## 列表与块快捷键

| 快捷键         | 功能     | macOS       |
| -------------- | -------- | ----------- |
| `Ctrl+Shift+B` | 无序列表 | `⌘+Shift+B` |
| `Ctrl+Shift+O` | 有序列表 | `⌘+Shift+O` |
| `Ctrl+Shift+Q` | 块引用   | `⌘+Shift+Q` |
| `Ctrl+Shift+C` | 代码块   | `⌘+Shift+C` |

## 全局快捷键

| 快捷键         | 功能       | macOS       |
| -------------- | ---------- | ----------- |
| `Ctrl+S`       | 保存文件   | `⌘+S`       |
| `Ctrl+O`       | 打开文件夹 | `⌘+O`       |
| `Ctrl+N`       | 新建文件   | `⌘+N`       |
| `Ctrl+B`       | 切换侧边栏 | `⌘+B`       |
| `Ctrl+Shift+T` | 切换主题   | `⌘+Shift+T` |
| `Ctrl+F`       | 查找       | `⌘+F`       |
| `Ctrl+Z`       | 撤销       | `⌘+Z`       |
| `Ctrl+Shift+Z` | 重做       | `⌘+Shift+Z` |
